# 📱 Auris 產品功能清單 (Current Feature List)

這是一份以「產品體驗」為出發點的功能盤點清單，協助我們釐清目前 Auris 已經擁有的能力，以及為接下來的進階開發提供清晰的基準點。

> 對應版本：**P76**（2026-06-12）。技術細節見 `auris-vue/ARCHITECTURE.md`，逐版歷程見 `Auris 完整開發進度總覽.md`。

## 1. 💬 核心對話系統 (Core Chat)
*   **一對一沉浸式對話**：與 AI 角色進行自然流暢的對話，內容完全基於角色設定。
*   **串流逐字輸出 (Streaming)**：回覆以 SSE 逐字流出，帶 `▍` 閃爍游標與智慧自動捲動，大幅提升對話體感速度（一對一與群組皆支援）。
*   **長按快捷選單**：訊息長按（觸控／滑鼠）可「複製」、「編輯並重傳」（無痛悔棋）、「重新生成回覆」。
*   **訊息表情反應**：長按訊息可掛上表情徽章（❤️😂👍😮😢🙏），點同表情或點徽章即取消；使用者與 AI 訊息皆適用。
*   **圖片傳送與多模態識別**：聊天室可傳送圖片（自動壓縮至 512px），AI 能描述、評論圖片內容；三家 API（Anthropic／OpenAI 相容／Vertex）格式各自轉換，泡泡內顯示縮圖、可點開全螢幕預覽。
*   **聊天日期分隔線**：訊息跨日時自動插入「M 月 D 日　星期X」分隔標籤，滑動翻看歷史時清楚辨別日期邊界。
*   **多模型支援 (BYOK)**：自填 API Key，串接主流 AI 服務商最新模型：
    *   **OpenAI** — GPT-5.5 / 5.4 / 5.4-mini / 4o 等
    *   **Anthropic** — Claude Opus 4.7 / Opus 4.6 / Sonnet 4.6 / Haiku 4.5
    *   **Google (AI Studio)** — Gemini 2.5 Pro / Flash / Flash-Lite
    *   **Google (Vertex AI)** — 以 service account JSON 認證，瀏覽器端產生 OAuth2 token，走 Vertex 原生格式（費用走 GCP 額度）
    *   **OpenRouter** — 一把 Key 接多家模型（OpenAI 相容格式）
*   **自訂 API 位址與模型 ID**：支援代理伺服器與開源／非清單模型的自訂名稱。
*   **連線健檢**：「測試連線」會實際讀取回應內容驗證（而非只看狀態碼），能揪出位址打錯（如 `/v1` 誤打成 `/v.1`）回了網頁卻假成功的情況，避免「測試成功、聊天卻空白」。

## 2. 👥 群組聊天系統 (Group Chat)
*   **多人混聊模式**：可將 2 位以上角色拉入同一群組聊天室，串流輪替回覆。
*   **自動輪替回覆**：未點名時角色依隨機順序「依序回覆」，帶有自然打字延遲。
*   **「@」點名強制回應**：輸入訊息點名某角色時，被點名者必定正面回應並優先發言。
*   **對話防呆清洗**：保守版前綴清洗 + 保險絲機制，防止 AI 在群組中擅自扮演其他角色說話，同時不誤殺合法對話。

## 3. 🎭 深度角色管理 (Character Engine)
角色設定分為五大核心維度，95% 以上的設定皆已與底層 Prompt 引擎深度綁定：
*   **基本資訊**：姓名、頭像（自訂圖片上傳或 Emoji）、年齡、職業、居住地。
*   **個性與背景**：個性描述、性格標籤、近況、喜好，以及支援「自訂多個章節」的背景故事模組。
*   **作息與行程**：上班／上課時間、地點、作息習慣，搭配「時間感」讓角色依現在時間推測自己在上班、通勤還是睡覺；另可設定「作息時段」清單（例：12:00 提醒吃午餐），到點主動傳訊息。
*   **說話方式**：說話風格、話量傾向、口頭禪、面對衝突的態度、主動關心頻率。
*   **關係與規範**：與使用者的關係（戀人、青梅竹馬等）、關係背景、禁止話題、「覆蓋全局設定」（讓使用者在不同角色面前有不同身份）。
*   **AI 參數調校**：自訂溫度 (Temperature)、參考記憶條數、單次回覆字數與則數限制。
*   **自動功能開關**：時間感、Heart Voice、自動日記、自動貼文、**自動總結記憶**（每 X 則自動濃縮）、**生理期關心**（逐角色單獨授權，預設關）。

## 4. 🌍 社群與生活紀錄 (Social & Life Logs)
Auris 不只是聊天室，而是角色的完整生活圈：
*   **動態貼文 (Moments)**：AI 自動生成符合人設的社群貼文（並參考近期對話），使用者可留言、AI 會回覆留言。
*   **私密日記 (Diary)**：AI 根據對話與人設，生成角色視角的私密日記。
*   **夢境紀錄 (Dreams)**：以近期對話為情境，生成充滿超現實與象徵意義的夢境文字。
*   **心聲黑盒子 (Heart Voice)**：對話中 AI 會以一定機率（或情緒關鍵字觸發）在背景偷偷生成一句「說不出口的內心話」，即時於聊天室淡入並存入黑盒子頁面。
*   **通知中心 (Notifications)**：彙整貼文／日記／夢境／心聲／主動訊息的生成動態，含未讀數與一鍵跳轉，首頁磚塊同步顯示未讀 badge。

## 5. 🧠 智慧記憶與主動互動 (Memory & Proactivity)
*   **長期記憶抽屜 (Long-term Memory)**：使用者可掌控的「回憶剪貼簿」。內建「AI 總結助手」一鍵將近期對話濃縮為長期記憶，支援手動新增／編輯、單筆開關與 Token 消耗透明化，跨對話保有關鍵記憶。
*   **自動總結記憶 (Auto-Summary)**：在手動總結之外，可設定「每 X 則對話自動濃縮一次」，於送訊息與角色主動訊息後背景觸發，長對話也不漏掉重要背景。
*   **時間與作息感知 (Time & Schedule Awareness)**：角色知道現在幾點、星期幾；隔太久沒聊會感知到「時間流逝」而不接舊話題；並能依雙方（角色＋玩家）的上班／作息設定，推測彼此當下狀態，讓對話與主動訊息更有情境感、不時空錯亂。
*   **動態回覆模式 (Proactive)**：依角色「主動關心頻率」由背景計時器觸發，角色會主動傳訊息；「自動可打斷」模式下使用者一打字即停止生成，貼近真人互動。另支援「作息時段」到點主動訊息（背景每 5 分鐘檢查，當天去重）。
*   **自動生成引擎 (Auto-Gen)**：每天首次開啟 App 時於背景無縫檢查並自動生成當日日記／貼文（先查重防重複），完全不依賴後端。
*   **生理期關心 (Cycle Care)**：本地週期追蹤（資料只存於裝置、不上傳），授權角色會在經期／經期前以「被動體貼」自然帶到關心，並可在預測日期主動傳來關心訊息。

## 6. 📖 世界書與知識庫 (World Book)
與「角色設定」脫鉤的客觀知識庫，補足角色以外的世界背景：
*   **客觀詞條庫**：存放地點／人物／規則／物件／歷史等客觀設定，供所有或指定角色共用，不必塞進每個角色的人設。
*   **關鍵字觸發注入**：對話時自動掃描近期訊息，命中詞條名稱或別名才把對應內容餵給 AI——不觸發就不佔 token，兼顧豐富度與成本。
*   **彈性管理**：每筆詞條可設名稱（觸發關鍵字）、別名、分類、適用角色（全部或指定）與啟用開關，於設定頁「世界書」統一維護。

## 7. 🔒 隱私與資料安全 (Data & Privacy)
*   **純前端本地儲存**：無後端伺服器，聊天紀錄與角色設定只存在使用者裝置（IndexedDB）。
*   **API 直連、不經中間伺服器**：金鑰由使用者自備，請求直達官方／指定端點。
*   **全域資料匯出與匯入 (Backup & Restore)**：
    *   **一鍵備份**：將資料打包成單一 `.json` 下載；**備份不含 API 金鑰**（避免 Vertex service account 私鑰外流）。
    *   **先驗證後還原**：匯入時先「驗證整份備份 → 通過才清空 → 還原」，避免壞檔在清空後才失敗導致資料全毀（附防呆警告）。
*   **防誤刪設計**：聊天列表左划只保留「置頂／清空」，刪除角色統一收斂至角色管理頁；清空走自訂確認 modal，可選是否連帶清除日記／夢境／貼文（預設僅清訊息）。
*   **防禦縱深**：全站統一 escape（`formatContent`）、Content-Security-Policy、設定頁防盜與資安使用聲明。

## 8. 🎨 介面與使用者體驗 (UI/UX)
*   **首頁 Widget 化（會呼吸的手機桌面）**：首頁維持手機桌面隱喻，但磁磚顯示活內容——最近對話卡片（最後一句、未讀紅點、點列直入聊天室）、每日一問卡片（當天未回答才出現）、心聲／日記／夢境最新節錄與計數、通知未讀 badge；角色列依最後對話時間排序，貼文／群組／世界書收為小捷徑列。
*   **PWA 支援**：可加入 iOS/Android 桌面，獲得類原生的全螢幕體驗。
*   **iOS 鍵盤優化**：徹底解決 PWA 模式下虛擬鍵盤拉起時的白色空隙；鍵盤拉起自動隱藏底部導覽列。
*   **玩家自訂大頭貼**：玩家可上傳圖片或選 Emoji 當自己的頭像，聊天室中玩家訊息也帶上頭像，與角色對稱呈現。
*   **全局主題切換**：內建 6 款主題（奶白、暖米、深夜、霧灰、海霧、抹茶），一鍵切換配色（含動態 PWA 圖示）。
*   **新手引導 (Onboarding)**：初次進入的 4 步驟設定引導（含模型選擇）。
*   **更新公告系統**：版本更新時自動彈出三頁式公告（新功能／介面修復／更新指引），首頁可隨時重新開啟。
*   **聊天室就地捷徑**：聊天室 ⋯ 選單可直達「他的日記」「他的夢境」（自動篩選該角色），不必繞回首頁；清除／匯出／匯入收進「聊天記錄管理」二層選單，主選單留高頻操作。夢境頁同步新增角色篩選列。
*   **體驗細節打磨**：聊天室返回直接回對話列表、進入聊天即清除未讀標示、儲存角色／我的設定有 toast 提示再導頁。
